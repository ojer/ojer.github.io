# 切换颜色方案

点击我的工具栏图标打开菜单, 选择颜色方案和滤镜

使用我的工具栏在 '亮色/暗色' 颜色方案之间切换

当你切换至一种颜色方案或滤镜时, 我会:

- 为你切换页面颜色

- 记住你为该域名所选择的颜色方案和滤镜

当你打开相同主域名的新标签页时, 我会:

- 将页面切换为你上次所使用的颜色方案和滤镜

# 颜色方案切换无效果

百分之 95 的可能性是该网址不否支持主题颜色方案切换.

但仍需要保持严谨, 打开一个支持切换主题颜色方案切换的网址尝试功能是否可用.

比如 MDN Web 文档:

https://developer.mozilla.org

如果你浏览的页面不支持颜色方案切换, 可以尝试开启 '智能反转' 滤镜

# 切换颜色方案后的显示效果与主题色相反

检查是否对该页面开启了反转滤镜

当页面支持主题颜色方案切换时, 使用反转滤镜会对颜色再次反转,

这样很好但是不推荐

# 数据丢失/未同步

桌面 Firefox 需要用户在同步设置中开启附加组件选项.

Firefox for Android 不会将数据与用户帐户同步.

扩展数据每 10 分钟或用户主动选择同步时同步一次.

或者在 "设置 > 同步" 或 Mozilla 帐户图标中手动触发同步.

最大可同步保存的数据大概在 400 多条.

准确来说, 最少可同步 404 条, 最多不超过 505 条.

最久未再使用的数据将最先被遗忘.
